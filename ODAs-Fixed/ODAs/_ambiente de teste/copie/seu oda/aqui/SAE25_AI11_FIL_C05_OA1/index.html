Para o professor

Encaminhamento metodológico

O conteúdo deste objeto digital serve para aprofundar o que está no livro, proporcionando mais uma maneira de fazer com que as crianças fixem os conceitos de semelhança e diferença. Dialogue com os alunos sobre o que eles perceberam no jogo – o que é identificar dois elementos semelhantes, e o que é identificar dois elementos diferentes – e como se faz para fazer essa identificação.
